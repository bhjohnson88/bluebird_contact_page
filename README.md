# Bluebird Portraits Contact Page

A minimal static contact page for Bluebird Portraits, designed for GitHub Pages.

## Deploy to GitHub Pages

1. **Create a GitHub repository** (if you haven't already):
   - Go to [github.com/new](https://github.com/new)
   - Name it `bluebird_contact_page` (or your preferred name)
   - Initialize with a README is optional

2. **Push this project to the repo**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Bluebird contact page"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/bluebird_contact_page.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**:
   - Open your repo on GitHub
   - Go to **Settings** → **Pages**
   - Under "Source", select **Deploy from a branch**
   - Choose branch `main` and folder `/ (root)`
   - Click **Save**

4. **Your site will be live at**:
   - `https://YOUR_USERNAME.github.io/bluebird_contact_page/`

## Custom Domain

To use a custom domain later:

1. Go to **Settings** → **Pages**
2. Enter your domain under "Custom domain"
3. GitHub will show the DNS records to add at your registrar (CNAME or A records)
4. Configure DNS, then wait for propagation (can take up to 48 hours)

## Replacing Placeholders

Replace the placeholder assets in `images/` when ready:

- `images/logo.svg` → replace with your logo (e.g. `logo.png`; update `index.html` src)
- `images/brand-mark.svg` → replace with favicon (e.g. `brand-mark.png`; update favicon link)
- `images/hero-banner.svg` → replace with hero image (e.g. `hero-banner.jpg`; update `index.html` src)
